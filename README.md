# agent-skills

Custom agent skills for installation via `skills.sh` / `npx skills add`.

## Available Skills

### [react-native-hidden-styles](skills/react-native-hidden-styles/SKILL.md)

Guidance for lesser-known and newer React Native styling techniques such as gradients, `boxShadow`, `filter`, `gap`, and `mixBlendMode`.

```bash
npx skills add kanmi-idris/agent-skills --skill react-native-hidden-styles
```
