# OpenCode Goal Skill

Shared skill that forces use of the OpenCode goal tools instead of handling goal state in plain text.

## Files
- `SKILL.md`: skill definition

## Behavior
Routes user intent to:
- `create_goal`
- `get_goal`
- `update_goal`

## Purpose
Use this when you want linked agents to strongly prefer the goal plugin tools whenever a user talks about OpenCode goal state.
