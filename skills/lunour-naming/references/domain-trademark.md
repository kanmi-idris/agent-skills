# Domain & Trademark Research Guide

This file provides practical guidance on checking domain availability and trademark risk for name candidates.

---

## Domain Strategy

### Priority Order for Domains

1. **`[name].com`** — Always the gold standard. Worth acquiring if under $5K for most startups.
2. **`get[name].com`** — Common and accepted.
3. **`[name].io`** — Widely used in tech.
4. **`[name].co`** — Acceptable for startups.
5. **`[name]hq.com`** or **`[name]app.com`** — Functional fallbacks.
6. **Country TLDs** (`.ai`, `.vc`, `.so`) — Only if the extension reinforces the brand.

### Domain Availability Rules of Thumb

- Single common English words: almost certainly taken on `.com`
- Two-word combinations: often possible depending on specificity
- Invented words: usually much higher availability
- If `.com` is taken, inspect ownership before rejecting the name

### Domain Red Flags

- Domain actively used by a competitor or similar business
- Domain held by a squatter with a steep asking price
- Domain redirects to something suspicious or brand-damaging

## Trademark Research

### Why It Matters

A domain can be available while the name is still trademarked. That can create serious legal and financial risk.

### Free First-Pass Checks

- **USPTO TESS** (US): https://www.uspto.gov/trademarks/search
- **EUIPO** (Europe): https://euipo.europa.eu/eSearch/
- **WIPO Global Brand Database** (International): https://branddb.wipo.int/

### What to Search

1. Exact name
2. Phonetic variants
3. Relevant product/service categories

### Risk Levels

| Situation | Risk Level | Recommendation |
|---|---|---|
| Name not found in searches | Low | Proceed, but still get legal review before launch |
| Name found in a different industry | Medium | Consult a trademark attorney |
| Name found in your industry | High | Avoid the name |
| Name is very descriptive | Medium | Harder to trademark and defend |

### Trademark Classes Often Relevant for Startups

- **Class 35**: Business services, SaaS, marketing
- **Class 36**: Financial services, fintech
- **Class 38**: Telecoms, messaging, social platforms
- **Class 42**: SaaS, software, technical development

### Key Principle

**A domain check is not a trademark check.** Always do both.

## International Checks

For globally-facing brands:

1. Run the name through translation tools for major languages.
2. Ask native speakers if the name has odd or negative connotations.
3. Check whether the pronunciation travels well.
